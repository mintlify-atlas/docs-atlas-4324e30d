---
title: Quickstart
description: Get a working Bookie instance running and add your first book in under five minutes.
---

This guide takes you from zero to a running Bookie instance with a book in your library.

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) installed and running
- [Docker Compose](https://docs.docker.com/compose/install/) (included with Docker Desktop)

<Steps>
  <Step title="Start Bookie with Docker Compose">
    Create a `docker-compose.yml` file with the following content:

    ```yaml docker-compose.yml
    services:
      bookie:
        container_name: bookie
        image: ghcr.io/sweatyeggs69/bookie:latest
        ports:
          - "5000:5000"
        volumes:
          - /path/to/config:/app/data
        environment:
          - SESSION_COOKIE_SECURE=false  # Required when accessing over HTTP
        restart: unless-stopped
    ```

    Replace `/path/to/config` with a local directory where Bookie will store your library and database.

    Then start the container:

    ```bash
    docker compose up -d
    ```

    Bookie will be available at [http://localhost:5000](http://localhost:5000) once the container is healthy. The health check polls `http://localhost:5000/` every 30 seconds, so the UI may take a few seconds to become available on first start.
  </Step>

  <Step title="Create your account">
    Open [http://localhost:5000](http://localhost:5000) in your browser. On a fresh install, Bookie redirects you to a setup page where you create the initial admin account.

    Fill in a username and password, then submit the form. You will be logged in automatically.

    <Note>
      Bookie does not have a separate admin role — the first account created is the only account needed for a personal instance.
    </Note>
  </Step>

  <Step title="Upload your first book">
    From the library view, click the **Upload** button (or drag and drop a file onto the page).

    Bookie accepts the following formats: EPUB, PDF, MOBI, AZW, AZW3, FB2, DjVu, CBZ, CBR, and TXT. The maximum upload size is 35 MB per file.

    Your book will appear in the library grid after upload.
  </Step>

  <Step title="Fetch metadata">
    Open the book's detail view by clicking its cover, then click **Fetch metadata**.

    Bookie searches Open Library, Apple Books, and Goodreads for a matching title and author. When results appear, select the correct entry to apply the metadata — title, author, publisher, description, and cover image — to your book.

    <Tip>
      For EPUB and PDF files, Bookie automatically extracts any metadata already embedded in the file before searching external sources.
    </Tip>
  </Step>

  <Step title="Send to your eReader">
    Once your book is in the library, you can send it directly to a Kindle or other eReader by email.

    See the [Send to eReader](/features/send-to-ereader) guide for full setup instructions including SMTP configuration.
  </Step>
</Steps>
