## What is Bookie?

Bookie is a self-hosted web application that gives you complete control over your eBook library. Run it in Docker, point it at your books folder, and get a fast, clean interface for organizing everything — with no subscriptions, no cloud dependency, and no tracking.

## How it works

Bookie is a Flask (Python) backend serving a React + TypeScript frontend, packaged as a single Docker image. Your library lives in a persistent volume on your host, and a SQLite database tracks all metadata. Nothing leaves your server unless you explicitly send a book to an eReader.

```
┌─────────────────────────────────────────┐
│           Your Docker host              │
│                                         │
│  ┌──────────────────────────────────┐  │
│  │         Bookie container         │  │
│  │  React UI  ←→  Flask API         │  │
│  │              ↕                   │  │
│  │          SQLite DB               │  │
│  └──────────┬───────────────────────┘  │
│             │ volume mount             │
│  ┌──────────▼───────────────────────┐  │
│  │      /path/to/config/            │  │
│  │      ├── books/                  │  │
│  │      ├── covers/                 │  │
│  │      └── booker.db               │  │
│  └──────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## Supported formats

| Format | Extension | Cover extraction | Metadata extraction |
|--------|-----------|-----------------|---------------------|
| EPUB   | `.epub`   | Yes             | Yes                 |
| PDF    | `.pdf`    | Yes (first page)| Yes                 |
| MOBI   | `.mobi`   | —               | —                   |
| AZW3   | `.azw3`   | —               | —                   |
| CBZ    | `.cbz`    | —               | —                   |
| CBR    | `.cbr`    | —               | —                   |
| FB2    | `.fb2`    | —               | —                   |
| DJVU   | `.djvu`   | —               | —                   |
| TXT    | `.txt`    | —               | —                   |

<Note>
  The maximum upload size is 35 MB per file. Metadata and cover search works best for EPUB and PDF files since they contain embedded information.
</Note>
