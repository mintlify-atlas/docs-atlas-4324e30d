---
title: Get a book
description: Retrieve a single book record by its ID.
---

`GET /api/books/{book_id}`

Returns the [Book object](/api-reference/overview#the-book-object) for the specified book.

## Path parameters

<ParamField path="book_id" type="integer" required>
  The unique ID of the book.
</ParamField>

## Response

Returns the [Book object](/api-reference/overview#the-book-object).

## Errors

| Status | Condition |
|---|---|
| `401` | Not authenticated. |
| `404` | No book found with the given ID. |

## Example

```bash
curl http://localhost:5000/api/books/42 \
  --cookie "session=<your-session-cookie>"
```
