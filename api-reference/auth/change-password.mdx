---
title: 'Change password'
description: 'Update the account password. Requires an active session.'
---

`POST /api/auth/change-password`

Changes the password for the currently authenticated account. You must provide the current password to authorize the change.

<Info>
  This endpoint requires authentication. Unauthenticated requests receive a `401` response.
</Info>

## Request body

<ParamField body="current_password" type="string" required>
  The current account password.
</ParamField>

<ParamField body="new_password" type="string" required>
  The new password. Must be between 8 and 256 characters.
</ParamField>

<RequestExample>

```bash cURL
curl --request POST \
  --url http://localhost:5000/api/auth/change-password \
  --header 'Content-Type: application/json' \
  --data '{
    "current_password": "oldpassword",
    "new_password": "newpassword1"
  }'
```

</RequestExample>

<ResponseExample>

```json 200
{"success": true}
```

```json 400
{"error": "Password must be at least 8 characters"}
```

```json 403
{"error": "Current password incorrect"}
```

</ResponseExample>

## Response fields

<ResponseField name="success" type="boolean">
  `true` when the password was updated successfully.
</ResponseField>
